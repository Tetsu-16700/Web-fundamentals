Crea un 'Tooltip'

Un Tooltip es un peque;o cuadro de informacion que aparece cuando pasas el cursor sobre un elemento en una interfaz grafica de usuario. Este cuadro ofrece detalles adicionales sobre la funcion del elemento, proporcionando asi una ayuda contextual. Por ejemplo, si pasas el cursor sobre un boton guarda cambios en una aplicacion , un Tooltip podria aparecer con el texto "Guardar", para clarificar que hace ese boton. Los Tooltip son utiles para mejorar le experiancia del usuario. ya que proporcionan orientacion sin requerir que se desplace a otra seccion para obtener informacion.

Requerimiento

Tu tarea es crear un Tooltip, un peque√±o elemento que proporciona informacion adicional cuando el usuario pasa el cursor sobre el elemento. Utilizaras JavaScript para manipular el DOM y crear este componente.

  • Elemento Objetivo : El elemento sobre el cual aparecera el Tooltip sera cualquier elemento que tenga un atributo `data-tooltip` definido.
  • Contenido del Tooltip :El valor del atributo sera el mensaje que aparecera en el Tooltip. Por ejemplo, un elemento con atributo `data-tooltip="Grabar"` mostrara el texto "Grabar" como contenido del Tooltip.
  • Posicionamiento : El Tooltip debe de aparecer encima del elemento objetivo.
  • Estilo :Sigue el dise√±o especificado en Figma para dar estilo al Tooltip
  • Interaccion : El Tooltip debe mostrarse cuando el usuario pase el cursor sobre el elemento objetivo y desaparecer cuando el cursor se mueva fuera del elemento. Pero no lo debe de hacer de inmediato. El mouse debe de mantenerse por encima del elemento objetivo antes de cumplirse los 300ms, el Tooltip debe mostrarse.

Requerimientos

Campos del formulario:
